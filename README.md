## 主要作品

**[DQA Lab Platform](https://github.com/tsaishengsheng-sketch/dqa-lab-platform)**  
環境測試實驗室管理平台 — FastAPI + React + SQLite  
[🚀 Live Demo](https://huggingface.co/spaces/sshengsheng/dqa-lab-platform)

- 5 國際測試標準（IEC 60068 / EN 50155 / IEC 61850-3 / IEC 60945 / DNV）78 條件、17 張資料表
- ISO 17025 合規：GUM 量測不確定度（Type A/B）自動計算，輸出 PDF 報告
- 設備校驗 & 維護排程 CRUD，校驗狀態即時監控
- AI 諮詢（Gemini + RAG）→ 排程確認 → 治具借出 → 報告下載，三模組全自動聯動
- **124 項 pytest 自動化測試** + GitHub Actions CI/CD
